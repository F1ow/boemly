# 🏎 Boemly Optimizations

If you want to get the optimum performance out of your app, add the following links to the head of
your app to pre-load all the needed fonts:

```jsx
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/ABCGintoNord-Bold.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/Inter-Bold.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/Inter-Medium.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/Inter-Regular.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/Inter-SemiBold.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
<link
    rel="preload"
    href="https://cdn.tree.ly/assets/v3/fonts/SpaceMono-Bold.woff2"
    type="font/woff2"
    as="font"
    crossOrigin="anonymous"
/>
```
